Importing the Models

To proceed with creating a simulation system, models must be imported into the library. Initially, there are three default folders available in every newly created project. They are:

  • Control
  • Environment
  • VECU

You can import the models in any of the folders above, but it is recommended that Plant models be imported into Environment folder, Controller models into Control folder, and VECU models into VECU folder. You can create sub-folders within those folders to allow further grouping of the models you want to import.

In this tutorial, we use the given structure to import an engine plant model into the Environment folder and a controller model in the Control folder in the library. The models are available as *.slxip formats. These models can be found in the COSYM demo data location Simple SiL Simulation.

To import the models

  • Right-click on the Environment folder to open the context menu and click to import a model into the folder.
  • The "Import Model" dialog box is displayed.

Import dialog box

  • Click Browse to select the provided sample models from the COSYM demo data location and select the model, MiL_Three_Cyl_7_Plant_.slxip.
  • Click Import to import the model into the library.

Proceed with the same steps to import the model, MiL_Three_Cyl_7_ECU.slxip into Control folder.
