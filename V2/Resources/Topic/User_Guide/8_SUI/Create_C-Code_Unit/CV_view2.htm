Calibration Variables of C-Code Models

The "Calibration Variables" view helps to add the calibration variables into a C-Code model.

This view is available for C-Code models only.

The following fields are shown in the "Calibration Variables" view.

  • Name
    Name of a calibration variable.
  • Type
    Displays the data type of a calibration variable.
  • Value
    Value of a calibration variable.

To add a calibration variable in the "Calibration Variables" view directly

  • Click .
  • A new calibration variable is added with the default name starting with "CVariable".

To add a calibration variable in a folder

  • Click on the drop-down arrow in the icon and click .
  • A new folder is added with the default name starting with "Folder".
    You can also create the sub-folders in it and rename the folder by right-clicking on a folder and click . You can perform all the folder operations such as copy, paste, and delete. The copied ports are appended with "_Copy" at the end of the port names.

Note 

While creating or renaming a folder or sub-folder, only the special characters valid for Windows are allowed. It means that characters like \, /, :, *, ?, ‚Äù, <, >, and | are not allowed.

  • A new calibration variable is added with the default name starting with "CVariable".

To update a calibration variable

  • Double-click on any calibration variable.

Changing the data types in Calibration Variables

By default, a variable was created with the real64 type. Based on the type selected, the value fields are displayed.

  • Bool:
    A true or false value can be selected from the drop-down menu.

  • ENUM
    Assign the required value(s) in the '#' and 'Val' field.
  • Click Update Values.
  • Once you update the values, the assigned value(s) in the 'Val' field are displayed in the "Select value" drop-down menu.
  • Select the displayed value from the drop-down menu.

Note 

The internal data type selected by default for the enum is SINT64.
  • REAL32, REAL64, SINT8, SINT16, SINT32, STRING, UNIT8, UNIT16, UNIT32:
  • Select the option button to select the data type. Based on the data type selected, a table is displayed to enter the values. It is also possible to add/delete the rows/columns.

  • Click Update to update the modified changes.
