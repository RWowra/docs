Mapping the Processes

Processes mapping helps to configure the OS by mapping the processes into the relevant tasks.

Process panel in a system displays all the available processes. You can map a process(es) into a task by dragging and dropping it individually or multiple processes can also be mapped into the tasks. The mouse cursor displays only for the valid position in the task panel while dropping the process(es) manually.

To map the process(es) to the task panel

  • Perform one of the following steps below:
    • Select a process or hold <Ctrl> key and select the multiple processes from the process panel.
      <Or>
    • Select one or more tasks in the task panel and click icon
      and right-click.
  • A context menu is displayed.
  • Click .
    • The selected process(es) is(are) mapped into the selected task(s).
      <Or>
    • Drag and drop the processes into tasks.

Similarly, the mapped processes to the tasks can also be removed as described below:

To unmap the process(es) from the Task Panel

  • Perform one of the following steps below:
    • Select a process or hold <Ctrl> key and select the multiple processes from the task panel.
    • Click icon.
      <Or>
    • Right-click.
  • A context menu is displayed.
  • Click .
  • The selected process(es) is(are) removed from the task panel.

The points below are considered during processes/tasks selection or mappings:

  • You can either select the tasks or the processes at a time in the task panel.
  • While dragging and dropping of one or more processes into a task, the process(s) will be placed after the process where you hover the mouse.
  • When you select the process(es) in the task panel and process panel and click to map the process(es), then the selected processes place after the selected process in a task panel. If multiple processes are selected in the task panel, then the processes place after the last selected process among all the selected processes in the task panel under each task.

Note 

You can also remove a process from the task panel by clicking icon on the relevant process.

Note 

Auto mapping feature in the Deployment view also helps you to configure the OS. Refer to Auto Mapping for more details.
