Importing the Models

To proceed with creating a simulation system, models must be imported into the library. Initially, there are three default folders available in every newly created project. They are:

  • Control
  • Environment
  • VECU

In this tutorial, the COSYM GRT (*.slxip) and FMU (*.fmu) models are used to import the models. These models can be found in the COSYM demo data location Simulation Models.

To import the models

  • Right-click on the Control/Environment/VECU folder to open the context menu and click to import a model into the folder.
  • The "Import Model" dialog box is displayed.

Fig. 57: Import dialog box

  • Click Browse to select the provided sample models from the COSYM demo data location and select the model, "SMod_AllDataTypes_cosym.slxip".
  • Click Import to import the model into the library.

In the same way, also import the sample testdata.fmu model into COSYM which is available at SiL\ATSModels\FMU\ folder in the COSYM demo data location.

Once the model is imported, update the variables accordingly as shown in the Fig. 58. Refer to Calibration Variables (FMU, COSYM-GRT target based MATLAB®/Simulink® and LCO C-Code Models) section for more details.
