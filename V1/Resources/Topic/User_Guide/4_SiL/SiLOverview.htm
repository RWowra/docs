SiL Overview

The software-in-the-loop (SiL) is a testing process where a software representation of the real electronic controller unit (ECU) is used. This software representation is called virtual electronic controller (VECU).

The interfaces of the plant model and rest bus simulation are connected with software interfaces for the signals, message buses and protocols. In the virtual ECU, the hardware pins and bus connectors of the physical ECU are replaced by software interfaces.

Fig. 6: SiL setup

Using a virtual ECU in SiL test systems, enables early validation at system level. The network of multiple controllers can be setup completely in software.

To enable this, the software stack of a controller has to be represented by models and algorithms (see Fig. 7).

The model depth, integration depth, and complexity of the software elements in the virtual ECU depend on the required accuracy and use case coverage of the system under test.
