Viewing the Simulation Result

It is necessary to install ETAS MDA visualize the signals of the generated mf4 files that contain the simulation result. COSYM supports V7.2.11 and higher version of V7.2.XX where X is the version number. And additionally, COSYM supports V8.5.3 too.

Please contact ETAS Sales Manager to install MDA.

To configure the Datalogger

  • Go to "Datalogger" tab in ETAS Experiment Environment (ETAS EE) and select the ICE_nEng signal of the plant model from the Workspace Elements.
  • Drag and drop the signal to Datalogger.

To download the models in ETAS EE

  • Go to Experiment menu in ETAS EE and click Download → SiL. Wait for the message, Model Download completed successfully logged in the "Application Log" window in ETAS EE.
  • The model is downloaded in ETAS EE.

To activate the datalogger

  • Check the checkbox, "Active" (shown as in the Fig. 20) to activate the datalogger.
  • Click Start (shown as in the Fig. 20) to start the datalogger.

Fig. 20: Datalogger in ETAS EE

To start and stop the simulation

  • Set the simulation end time as -1 seconds on the "Adaptive time" field in the tool bar of ETAS Experiment Environment (ETAS EE).
  • Click Start Simulation in Experiment menu. Then a message Simulation started is displayed in the Application Log.
  • Click Stop Simulation to stop the simulation process.
  • Click Disconnect to disconnect the simulation process.

To view the simulation result

The Application log in the ETAS EE displays the location of the simulation logger in the Fig. 20.

  • Double click on the <File_name>.mf4 file.
  • Opens the simulation result log file from the MDA V7.2.11.26.
  • Select the "ICE_nEng" signal of the plant model to plot the simulation results for the engine speed.
  • Click OK.

Fig. 21: Variable selection in MDA

  • Select the default "New Oscilloscope" display window and click OK.

  • Select the axis type as "one axis per unit" and click OK.

  • The result is plotted in the oscilloscope.

Please note that, the simulation result above is viewed in MDA V7.2.11.26. The procedure to view the result and the graph may slightly vary in other versions.

Fig. 22: Simulation logger for ICE_nEng signal of the plant model

The plotted simulation results above show the values of the engine speed. The simulation starts off with engine speed at approximately 2000 rpm, following which, the engine speed reduces and settles at its idle speed of 800 rpm. This can be attributed to the zero input at the user accelerator pedal.
