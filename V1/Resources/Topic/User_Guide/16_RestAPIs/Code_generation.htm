Generating Client Code

Client generators are currently available for Java, JavaScript, C#, and Python clients. Generate client feature generates the client code.

Once client code is generated in the desired supported language, you can use it to interact with COSYM application. For e.g., a complete end to end flow of creating project, adding models and building the system can be performed using these generated client code. Refer to Simple SiL Simulation Automated with REST APIs.

To generate the client code

  • Go to Help menu → API → Generate client... on the project screen or project dashboard.
  • The "Client generation" dialog box is displayed.
  • Select the language of the client code which is to be generated from the Type drop-down menu. By default, Java is selected.
    If you select "C#'" for client code generation, then refer to Open Source DLLs for C# for more details.
  • Specify the path in the Location field or click Browse to select the path to the client code.
  • Click Export.
  • Client code is exported to the specified location.

Refer to README.md to use the generated code.

Open Source DLLs for C#

When you generate a client code in "C#" language and if you find JsonSubTypes.dll and RestSharp.dll missing DLLs, then you can add those DLLs from the <COSYM Installed location>\Software\OpenSourceDLL folder to proceed further.

Host URL configuration

The default port used for the COSYM application is 8181. Host Configuration setting for different client is shown below.

  • C-Sharp: Refer to Simple SiL Simulation Automated with REST APIs.

  • Java: Open the generated \src\main\java\io\swagger\client\ApiClient.java and update host as "http://localhost:8181/cxf/cosymserver" which is shown below.

  • Java Script: Open the generated src/ApiClient.js and update host as "http://localhost:8181/cxf/cosymserver" which is shown below.
