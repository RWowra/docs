Adding Model Instances into a System

You can add an instance of a model only when a system is opened and selected in the project dashboard.

Model instances can be added with the following methods:

  • Add To System
  • Drag and drop the models

As soon as a model instance is added to a system from the library, the instance is also added into the "Units" folder in systems in the project panel.

When you double-click on a model from the "Units" folder in the systems, a tab is opened in the model dashboard, but when you double-click on a CAN, network restbus models the corresponding editor will be opened for modification.

Note 

Multiple instances of an FMU can be added into a system whereas only one instance of an LMD, SCOOP-IX, or VECU FMU can be added.

Note 

If a Simulink model is added as an lmd, the same model cannot be added to a system as SCOOP-IX or vice-versa.
