Parameter Assistant

The "Parameter Assistant" view in a system tab represents the aggregated view of categories and workpackages of all the model instances. The parameters of different model instances are available in the respective workpackages inside the respective categories. The parameters are displayed only for the Active parameter files.

COSYM allows to create categories, workpackages, and adding parameters to the workpackages only in the Parameter Assistant view of a model. In a system, it is possible to edit the "Value" and the "Comment" fields only, if the parameter file is an Active and a Cloned Copy.

To view the Parameter Assistant view in the system tab

  • Click on the Parameters in a system tab to navigate the Parameter Assistant view.
  • The fields in the "Parameter" view are displayed as shown below.

The parameters in a system

The parameters of different model instances are available in the respective workpackages inside the respective categories. When you select a workpackage from the "Category or Wokpackage" panel, all the parameters that are assigned to the workpakage are displayed in the system tab under the respective model instances.

The properties of a parameter are listed below. All these properties are non-modifiable except the "Value" and the "Comment".

  • Name
    Name of a parameter
  • Model Path
    The path of a parameter
  • Value
    Value specified for the parameter which can be modified
  • Min Value
    Minimum range of a value
  • Max Value
    Maximum range of a value
  • Type
    Parameter type
  • Comments
    Comments if any
Edit the Values/Comments fields in a system tab

COSYM allows to edit only the Value and Comment fields in a model instance if the parameter file is an Active and a Cloned Copy.

To edit a workpackage

  • Select a workpackage.
    All the parameters that are assigned to a workpakage are displayed in the system tab under their respective model instances.
  • Double-click on a parameter.
    The "Edit parameters" dialog box is displayed.

Fig. 176: Edit Parameters dialog box

  • Edit the value or(and) comment fields in a parameter.
  • Click Save to update the changes.
