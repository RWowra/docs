Conversion

A conversion model helps to modify the value of an inport of a model by applying certain rules to it. When conversion model is applied, the new value is assigned to an inport. The workflow of a conversion model is shown in the Fig. 148.

Fig. 148: Conversion model workflow

COSYM allows adding multiple instances of a signal conversion model into a system.

A signal conversion can be applied in the "Parameters" view of a conversion model.

Refer to To create a model for creating a conversion model. Along with the default fields of creating a model, additionally it is necessary to specify the "Parameter file" in the conversion model as given below.

  • Specify the location of the parameter file (*.mpa, *.dcm, or *.cdfx) in the "Parameter file" field.
    <Or>
    Click Browse to select the destination.
  • If the parameter file is not selected, then COSYM creates a default parameter file (model name_defaultparameter.mpa).
  • Conversation model is created and displayed in the model dashboard.
