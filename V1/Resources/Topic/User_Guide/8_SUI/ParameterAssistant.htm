Parameter Assistant View

The "Parameter Assistant (PA)" view helps to organize the parameters which are present in the selected model or model instance. It helps to add the categories and workpackages to structure the parameters as per your requirement for the calibration of a model.

In a model, it is possible to add or remove the categories, workpackages and assign the parameters to the workpackages.

In a model instance, if the parameter file is an active and a cloned copy, then the parameter values and the comments can be edited.

The parameter assistant takes the parameters values from the active parameter file. If the active parameter file is original parameter file, then the parameter assistant is in the read-only mode. In this mode, the values and comments cannot be edited. It is possible to edit the values and comments only on the working copy of the parameter file.

The page navigation feature is enabled when more parameters are available in a parameter table. By default, 25 rows can be viewed in a page. It allows you to scroll-up or scroll-down in a page. The number of rows can be changed to 50/100/500 from the drop-down menu in the page navigation. This drop-down menu appears when rows exceeds 25 onwards.

Note 

Cloning of the parameter file is possible only in the model instances.

To view the parameter assistant view

  • Open a model dashboard (To open a model dashboard for a model/model instance).
  • Click Parameter Assistant to navigate the PA view.
  • The fields in the parameter assistant are displayed.

Creating a Category

COSYM allows to create a category only in a model. The "Create Category" feature is enabled only after selecting a parameter file (<filename>.mpa or <filename>.cdfx) from the "Parameter file selection" drop-down menu.

Follow the steps below to create a category

  • Perform one of the following steps:
    • Select the relevant parameter file (<filename>.mpa or <filename>.cdfx) from the Parameter file selection drop-down menu.
      Category option is enabled.
    • Click on drop-down menu and select Category.
      <Or>
    • Right-click on the left side panel.
    • Click on on the opened context menu.
  • The "Create Category" dialog box displays.
  • Type the category name in the "Name" field.
  • Click Create.
  • A new category is created.

Creating a Workpackage

COSYM allows to create a workpackage only in a model under a category. You can create one or more workpackages under a category.

To create a workpackage

  • Perform one of the following steps
    • Select a category to create a workpackage.
      The workpackage option is enabled.
    • Click on the drop-down menu and select Workpackage.
      <Or>
    • Right-click on a category.
    • Click Create Workpackage on the context menu.
  • The "Create Workpackage" dialog box is displayed.
  • Type the workpackage name in the "Name" field.
  • Click Create.
  • A new workpackage is created under the selected category.

Assigning Parameters to a Workpackage

COSYM allows to assign the parameters to a workpackage in a model only.

To assign parameters to a workpackage

  • Select a workpackage to assign the parameters.
  • The Add/Remove parameters button is enabled.
  • Click Add/Remove parameters.
  • The "Add/Remove parameters" dialog box is displayed.

Fig. 132: The "Add/Remove parameters" dialog box

  • Select a parameter and click .
    <Or>
    Hold <Ctrl> key and select the multiple parameters and click to select multiple parameters.
    <Or>
    Click to select all the available parameters.
  • The parameter(s) is(are) moved to the "Mapped parameters" section.
  • Click Save to save.
  • The selected parameter(s) is(are) assigned to the selected workpackage.

Renaming a Category or Workpackage

To rename a category or workpackage

  • Right-click on a category or workpackage.
  • A context menu is enabled.

  • Click Rename.
  • The "Rename" dialog box is displayed.
  • Specify the new name in the "Name" field.
  • Click Update.

Delete a Category or Workpackage

To delete a category or workpackage

  • Right-click on a category or workpackage.
  • A context menu is enabled.

  • Click Delete.
    <Or>
    Click on the left side panel.
  • The "Delete Confirmation" dialog box displays.
  • Click Delete.

The Parameters in a Workpackage

The Fig. 133 shows the parameters which are assigned to a workpackage.

Fig. 133: Parameters in a Workpackage

The properties of a parameter are listed below. All these properties are non-modifiable in a model or model instance except the "Value" and the "Comment" fields.

  • Name
    Name of a parameter.
  • Model Path
    The path of a parameter.
  • Value
    Value specified for a parameter. The "Value" can be modified.
  • Min Value
    Minimum range of a value.
  • Max Value
    Maximum range of a value.
  • Type
    Parameter type.
  • Comment
    Comments if any.

Edit the Value/Comment Fields in a Model Instance

COSYM allows to edit only the "Value" and "Comment" fields in a model instance if the parameter file is an active and a cloned copy.

To edit a parameter

  • Double-click on a parameter in a model instance.
  • The "Edit parameters" dialog box is displayed.

Fig. 134: The "Edit Parameters" dialog box

  • Edit the "Value" or(and) "Comment" fields in a parameter.
  • Click Save to update the changes.

Note 

When you build a system, an experiment is created with the currently active parameter file of each model. The parameter assistant view is only a visualization for the usability purpose.
