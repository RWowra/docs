Ports of C-Code Model

The "Ports" view helps to add the inports and outports into a C-Code model. It is possible to add a single port or multiple ports in the "Ports" view. You can create maximum of 9, 999 ports at a time and it takes a few seconds to create maximum ports. You can import or export the ports in the Ports view.

Fig. 118: The "Ports" view in a C-Code model

Fields of the Ports view in a C-Code model

The following fields are shown in the Inports or Outports.

  • Name
    Name of an inport/outport. You can edit the name, if required.
  • Type
    The data type of an inport. It supports the data types such as boolean, enum, real32, real64, sint8, sint16, sint32, uint8, uint16, and uint32 . By default, the real64 data type is selected.
  • Delete

These added ports are shown in a hierarchical tree structure in the Connections viewer.

To rename a folder

  • Right-click on a folder and click .
    You can perform all the folder operations such as copy, paste, and delete. The ports are appended with "_Copy" at the end of the port names.

To update a port

  • Double-click on any port.
  • The "Port Details" dialog box is displayed as shown in the Fig. 119. By default, a port is created with the real64 type.

Fig. 119: The "Port Details" dialog box

  • You can rename the "Name" of a port.
  • Select the data type from the "Type" drop-down menu.
    If 'ENUM' data type is selected, you can assign the values as described inTo change the EnumUM data type in ports.
  • Click Update to save the changes.

To change the EnumUM data type in ports

When you select the enum data type from the drop-down menu, the dialog box is updated with the "Value" field where you can assign the required value(s) in the '#' and 'Val' fields.

  • You can add/remove the columns if required.
  • The Size (y) is updated based on the addition/deletion of the columns.
  • Assign the required value(s) in the '#' and 'Val' fields.
  • Click Update.

Fig. 120: The "Ports Details" for enum data type

To add multiple ports

  • Click on the drop-down arrow in the icon and click .
    The "Add Inports" or "Add Outports" dialog box is displayed based on your selection.

Fig. 121: The "Add Inports" dialog box

  • Specify the number of ports to be created in the text box.
    You can create maximum of 9, 999 ports.
  • Mention the name (prefix) to be specified for ports.
    By default, it takes the prefix as In for the Inports and Out for the outports.
  • Select the data type of the port from the "Type" field.
  • Click Add.
    The specified ports are added.

Refer to Port Numbering in a C-Code Model for the limitations of adding ports.

Importing and Exporting of Ports

The "Import" and "Export" feature helps to import or export of ports in a C-Code model. It is possible in the model only.

To import the ports

  • Perform one of the following step to import the ports.
    • Click .
      <Or>
    • Right-click on a port.
      A context menu is displayed
    • Click Import.
  • The "Import Ports" dialog box is displayed.

Fig. 122: The "Import Ports" dialog box

  • Specify the location of the *.xml file to import the ports or click Browse to select it.
  • Check the checkbox, "Ignore inports" to exclude the import of inports.
  • Check the checkbox, "Ignore outports" to exclude the import of outports.
    You can either check "Ignore inports" checkbox or "Ignore outports" checkbox, but not the both. If both are selected, then Import button is disabled.
  • Click Import.
  • The "Replace or skip port(s)" dialog box is displayed if the port(s) in the selected file is already available.
  • Click either "Replace the port(s) in the destination" to replace the existing port(s) or "Skip these port(s)" to ignore the port(s).
    • If all the ports in a model are of same name and same data type, then COSYM does not overwrite the ports.
    • If a port with the same name exists in a model with the different data type, then COSYM overwrites the port and removes the associated connection.
  • The ports are imported.

To export the ports

  • Perform one of the following step to export the ports.
    • Click .
      The "Export Ports" dialog box is displayed.
      <Or>
    • Right-click on a connection.
      A context menu is displayed.
    • Click Export.
      The "Export Ports" dialog box is displayed.

  1. Specify the location to export the ports or click Browse to select it.
  2. Check the checkbox, "Export inports" to export the inports and(or) "Export outports" to export the outports. By default, both the checkboxes are selected. If you want to export either inports or exports, then selected the respective checkbox only.
  3. Click Export.
  4. The selected ports are exported into a *.xml format in the specified location.

To export the selected port(s)

  • Right-click on a connection or hold <Ctrl> key and select the connections to select the multiple connections.
  • A context menu is displayed
  • Click Export selected port(s).
  • The "Export Ports" dialog box is displayed.
  • Select the location to export the ports.
  • Click Export.
