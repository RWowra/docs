External Files View

The "External files" view in a model dashboard helps to add only the files of type which are listed below or you can give the reference to the source files of any other model in a project.

  • Include files (a folder which contains *.c or *.h files)
  • Object files (*.o files)
  • Source files (*.c or *.h)
  • Libraries (*.a files)
  • Reference files (any source files)

It also allows to add a folder/sub-folder which contains the header files/source files/library files/reference files (*.c, *.h, *.o, *.a) only. When a folder/sub-folder is added with any of these files, the files are stored in the respective directories and can be visualized in the respective folders in the "Properties" tab of a model/model instance. An error message is shown, if a folder contains any unsupported file formats.

When you are adding the external files, make sure that there is no duplication of external files. For example, if the file <filename>.h is already available in the Include folder and if you try to add it in the Sources folder as well, then COSYM displays an error message and does not allow to add it again.

Note 
    For a consistent workflow, it is not recommended to combine the manual operations and operations via COSYM GUI/rest APIs together.
    For example, you cannot copy/paste the files manually to the folder and perform some GUI operations in order to add these file information to COSYM metadata. Therefore, it is necessary to perform every operation via GUI/rest APIs for consistent results.

 

Note 
  • External files view will not be available for FMUs.
  • The files/folders/references cannot be added/deleted in the model instance level. It is possible only in a model level.

To navigate the "External files" view

  • Open a model dashboard (To open a model dashboard for a model/model instance).
  • Click External files to navigate to the "External files" view.
  • The fields in the "External files" view are displayed.

Fig. 135: The "External files" view

Adding External Files

To add an external file

  • Click the drop-down menu and select .
  • A Windows Explorer is opened to select a file(s).
  • Select file(s) and click Open.
  • The selected file(s) is(are) stored under the respective directories in the "External Files" view.

Adding Folders

To add a folder/sub-folder

  • Click the drop-down menu and select .
  • A Windows Explorer is opened to select a folder/sub-folder.
  • Select a folder/sub-folder and click Open.
  • The files in the selected folder are stored under the respective directories in the "External Files" view as shown in the .
