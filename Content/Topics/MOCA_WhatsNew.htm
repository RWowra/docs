What's New in ASCMO-MOCA V5.14

General

  • ASCMO can now import Parquet files (Bosch variant).

  • ASCMO's p-code can now be started in MATLAB® R2023b (latest supported version).

ASCMO-MOCA

  • In addition to parameters, bounds can now be imported from another open MOCA instance.

    Import Bounds Only

    Import Parameters and Bounds

  • The map smoothing dialog now allows to display 2D cuts of maps (X-Z and Y-Z view).

    Parameter Smoothing

  • MOCA now allows to display the evaluation on a reference parameterset. Residuals and predictions can now be shown for two parametersets in one plot.

    Element Configuration

  • Matrix parameters can now depend on another parameter, e.g. to optimize only a single value of a larger vector/matrix: [1 2 %aScalar% 3].

    Create/Edit Parameter

  • Plot cursors have been improved to show the number of points above/below the cursor in the context menu.

    Visualization Step

  • Plots now have the same width regardless of the number of signals, so cursors are aligned in multiple plots.

  • MOCA now allows to export the function as DLL for MOCA Runtime (license management can be added by the user).

    File Menu

  • The Pause toolbar button now also pauses the evaluation of the function, not just the external model evaluation.

    Toolbar

  • Enumerated scalars, enumerated curves as axes, and enumerated group axes can now be imported from a DCM file and the associated A2L file.

    Available Types of Parameters

  • When importing a DCM file with variants, the user can select a specific variant to import. When exporting a variant, (next to the Exportbutton)>Patch Parameter or Variant in Existing DCM File can be used to select an existing DCM file and replace a specific variant.

    Import and Export DCM with Variants (Bosch only)

  • New Visualization: 3D Signal Plot. View multiple signals in a 3D plot and marked data in other 2D plots.

    Element Configuration

    Visualization

  • Search/Filter field added to the parameter table.

    Parameters Step

  • Improved table editing. Typing overwrites a cell, double-clicking a cell selects the whole content, better selection colors, ...

    Parameters Step and much more

  • Visualization elements can now be copied and pasted.

    Visualization Step

  • FMU V3.0 can now be simulated and various data types are supported (integer, single,…).
