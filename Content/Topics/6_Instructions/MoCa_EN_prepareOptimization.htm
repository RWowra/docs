Preparing an Optimization

Preparing an optimization contains settings in the "Optimization Options" area, "Optimization Criteria" area and "Local Constraints" area.

  1. In the "Optimization Options" area, do the following:

    1. Select an optimizer, see more information about Optimization Algorithms.

    2. In the "Number of Iterations" field, enter the desired number of iterations.

    3. Adjust the "Tolerance/Accuracy" value according to your needs.

    4. Adjust the "Finite Difference Factor" according to your needs, see Optimization Step or Step 6: Optimization for more information.

    5. If desired, increase the number of optimization runs in the "Multistart" field.

  2. In the "Optimization Criteria" area, do the following:

    1. In the first combo box, select the function output you want to optimize1 e.g., TorquePredict.

    2. In the second combo box, select the respective imported data channel2 e.g., Torque_Meas.

      The flat parameters results in a high RMSE. After the first optimization, the RMSE will be significantly reduced.

    3. If desired, add another optimization criterion.

  3. In the "Local Constraints" area, do the following:

    1. Click on Add a new Optimization constraint to add a local constraint.

    2. In the first combo box, select the channel or function node you want to limit.

    3. In the second combo box, select a comparison operator.

    4. In the third combo box, select the channel or function node to be used as limit.

      Note 

      You cannot directly enter a number <n> as limit. To use a number as limit, create a function node <node> with expression <n> and select <node> as limit. Alternatively, you can create a scalar parameter which is not optimized and use it as limit.

    5. If desired, create another constraint.
