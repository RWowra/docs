Options

The Options window contains the following elements:

Help Language

Here, you can select the language of the online help.

Advanced Settings

This activates the display of further settings.

Open GL

Setting for Open GL support. Possible selections:

Default (Hardware Basic)

If available, hardware support is used (default setting).

Software

Emulation by software.

Use this setting if you are experiencing problems such as blue screens or hidden points in the scatter plot. A restart of ASCMO is required.

Hardware

Hardware support is used.

CSV Export

Cell Separator: The character that separates the table cells when exporting to CSV format.

Decimal Separator: The decimal separator in a CSV export.

Simulink Version

Select the Simulink version you want to use with ASCMO-MOCA/ASCMO-MOCA Runtime. The available selections are Last Installed1 = the Simulink version most recently installed on your PC (not necessarily the latest version) and the Simulink versions installed on your computer.

Use Full Desktop Matlab

Note 

Available only if you started ASCMO-MOCA from within MATLAB/Simulink.

If activated, an external MATLAB/Simulink is used.

This is necessary, for example, if you are using a model that only runs on a 32-bit MATLAB/Simulink version, because ASCMO-MOCA is only compatible with a 64-bit MATLAB/Simulink version.

Autosave Iteration

If activated, a copy is automatically saved to %projectpath%\%project%.autosave.moca every minute during optimization and once after optimization. Useful if your computer is automatically restarted during optimization.

Reset

Resets all options to their default values.

OK

Applies your settings and closes the window.

Cancel

Discards your settings and closes the window.
