Creating and Editing a System Constant

To specify a system constant, proceed as follows.

  1. In the Parameters pane, go to the "System Constants" tab.

    If there is no system constant, or if no system constant is selected, only the "System Constants" table with a single line is visible.

  2. In the "Name" column of the "System Constants" table, click on the word New.

  3. Enter a name for the system constant and press <Return>.

    The system constant is created. The "Default Values for <system constant>" table appears.

  4. In the "Default Values for <system constant>" table, do the following:

    1. In the "Parameter" row, click on the words Add Parameter to open the parameters list.

    2. Select a parameter.

    3. In the "Default Value" column, enter the value you want to assign to the parameter.

      You have to enter a scalar value for all types of parameters. For non-scalar parameters, the same value is returned for each point.

    4. If desired, add further parameters to the system constant.

  5. If desired, add and specify further system constants.

You can return to the "System constants" tab and edit existing constants at any time.
