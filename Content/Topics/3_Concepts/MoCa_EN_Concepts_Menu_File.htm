File Menu

The File menu consists of the following entries:

New

Note 

Only available for ASCMO-MOCA.

Opens a new instance of ASCMO-MOCA where you can create and edit a new project.

Open...

Load an existing project (*.moca/*.moca_runtime). You can select the project in an open file dialog.

Recent Projects

Opens a list of recently edited projects.

Save

Saves the current project.

If the project has not yet been saved, an open file dialog opens where you can select the file name (*.moca/*.moca_runtime) and the file path.

Save As

Save the current project under a different name and/or location. You can specify the name and file path in an open file dialog.

Export to MOCA Runtime

Opens the Export to MOCA Runtime window, where you can specify the settings for the export to *.moca_runtime.

Show Sequence: The Optimization > Sequence tab can be shown or hidden in the MOCA Runtime project.

Allow Opening in MATLAB® (MOCA Runtime p-Code): Specify whether the exported project can be opened in MOCA Runtime SDK, i.e., started in MATLAB®. Projects encrypted for MOCA Runtime cannot be opened in MOCA Runtime SDK.

Split into MOCA Runtime and DLL file: The MOCA Runtime project file is split into two parts, a MOCA Runtime file and a DLL file. This can be used to attach a license to the MOCA Runtime project. When the project is opened again, the DLL must be in the same directory.

Export Function as FMU...

Exports your created function as an FMU file. You can specify the name and file path in an open file dialog.

Export Datasets

Opens the Export Datasets window, where you can specify the settings for the export to *.xls / *.xlsx, *.csv, *.dat, *.mf4, or *.mat.

  • Imported Data: Exports the original input data without evaluations.

  • Project Data with Evaluations: Exports the mapped input data, nodes, criterias, and evaluations.

  • Single File: Exports selected datasets to a single file. After clicking Export..., you can specify the file name and select the file path.

  • Multiple Files: Exports selected datasets to multiple files. After clicking Export..., you can specify the folder name and select the file path. The dataset names are used as file names.

  • Select File Format: Select from Excel (*.xls), Excel 97-2003 (*.xlsx), Comma Separated Values (*.csv), MDF3 Measurement Data Format(*.dat), MDF4 Measurement Data Format(*.mf4), or MAT (*.mat) file.

Options

Opens the "Options" window.

Close

Closes the current project and returns to the ASCMO-MOCA/ASCMO-MOCA Runtime start window.

Exit

Closes the current project and exits ASCMO-MOCA/ASCMO-MOCA Runtime completely.
