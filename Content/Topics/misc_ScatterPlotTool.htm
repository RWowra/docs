ASCMO-Scatter Plot

The standalone ASCMO-Scatter Plot offers the possibility to view n-dimensional data as a series of scatter plots without starting ASCMO-STATIC or ASCMO-DYNAMIC.

Using the standalone scatter plot tool

  1. In the "ASCMO-DESK" window, click on Scatter Plot.

    The "ASCMO Data Import" window and a file selection window open.

  2. In the file selection window, enter or select path and name of the file you want to import.

    You can import Excel (*.xls, *.xlsx) files and lists of comma-separated values (*.csv, *.txt).

  3. Click on Open.

    If the file contains several work sheets, the "Select Sheet" window opens.

  4. In that window, select the desired work sheet and click on OK.

    The content of the file is loaded into the "ASCMO Data Import" window. The data as well as the names of the measurement channels are displayed.

  5. In the "Available Data" list, select the measurement channels you want to plot.

  6. Click on Inputs.

    The variables are added to the "Inputs" list.

  7. Click on OK.

    The "Standalone Scatter Plot" window opens. See Scatter Plot Window for a description of the window elements.
