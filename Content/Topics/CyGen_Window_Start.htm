Start Window - ASCMO-Cycle Generator

The start window provides access to all the functions of the ASCMO-Cycle Generator. It contains the following elements.

File menu

  • New

    Opens another instance of the start window.

  • Options

    Opens the "Options" window where you can make user-specific settings.

  • Clear History

    Clears the entries under "Last Imports" and "Last Projects".

  • Start ASCMO Desk

    Opens the ASCMO-DESK window. In that window, you can start all ETAS ASCMO components installed on your computer.

  • Exit

    Closes the ASCMO-STATIC start window.

Plugins menu

  • Select Plugins

    Opens the "Plugins" window where you can select/deselect installed ASCMO plugins.

Help menu
Start Cycle Generator

Opens the ASCMO-Cycle Generator main window with a new, empty project.

Open ASCMO Project

Opens a selected ASCMO-Cycle Generator project.

"Last Projects" list

This list shows links to recently used ASCMO-Cycle Generator projects.
