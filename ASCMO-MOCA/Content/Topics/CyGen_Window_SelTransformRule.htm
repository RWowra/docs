"Select a conversion rule" Window

This window is opened with the Configure Conversion Rule option in the Conversion Rules menu of the ASCMO-Cycle Generator main window.

The "Select a conversion rule" window contains the following elements:

Conversion Rules list

This list shows the conversion rule templates (Simple Vehicle Model and External Conversion Rule) and the conversion rules in your project.

Note  

The External Conversion RuleExternal Validation Rule template can be used only in the P-code version of ETAS ASCMO (ASCMO_SDK license).

You can filter the list by entering a search term in the "Search conversion rules" field at the top of the list.

"Description" tab

This tab contains a description of the selected conversion rule and its configuration.

Note  

Read the information in this tab carefully.
"Configuration" tab

This tab contains all properties that can be set for the selected conversion rule.

Note  

You cannot edit the properties of the conversion rule templates (Simple Vehicle Model and External Conversion Rule). Use the Duplicate button to create an empty copy of a template, then edit the copy.

Simple Vehicle Model

  • "Conversion Rule Settings" area

    "Conversion Rule name" field

    Can be used to change the name of the conversion rule.

    The remaining content of the area depends on the type of the selected conversion rule. See the tooltips for information on the fields, options, etc.

  • "Input Mapping" table

    The "Inputs" column lists the input parameters for the conversion. In the "Cycle column names" column, select the name of the cycle columns that contain the respective parameters.

  • "Output Mapping" table

    The "Outputs" column lists the output parameters for the conversion. In the "Cycle column names" column, you can select existing cycle columns that will contain the respective parameters, or you can create new columns.

External Conversion Rule

  • "Conversion rule name" field

    Name of the conversion rule.

  • "Source" combo box

    This combo box shows the available source types for conversion rules, e.g., Matlab function.

  • "File path" area

    You can enter (in the "Conversion rule" field) or select (via the Browse button) a file that contains the external conversion rule. The allowed extension depends on the selection in the "Operation Mode" combo box.

  • "Input mapping" table

    The "Inputs" column lists the input parameters for the conversion.

    In the "Cycle column names" column, use the combo boxes to select the name of the cycle columns that contain the respective parameters.

  • "Output mapping" table

    The "Outputs" column lists the output parameters for the conversion.

    In the "Cycle column names" column, use the combo boxes to select existing cycle columns that will contain the respective parameters. You can create new columns via the New entry.

Delete conversion rule

Deletes the selected conversion rule. You can delete only one conversion rule at a time.

Duplicate

Creates a copy of the selected conversion rule or conversion rule template. Templates are marked with a gray background.

The copy of a template is empty, the copy of an existing conversion rule contains all settings of the original.

Select and close

This button selects the conversion rule and closes the "Select a conversion rule" window.

Cancel

Discards your settings and closes the window.
