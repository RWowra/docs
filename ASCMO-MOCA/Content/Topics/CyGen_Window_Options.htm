"Options" Window (ASCMO-Cycle Generator)

The "Options" window contains the following elements:

Help Language

This is where the language of the online help can be defined.

Advanced Settings option

At present, the ASCMO-Cycle Generator has no advanced settings.

Open GL

Setting for open GL support. Possible selections:

Standard

If available, hardware support is used (default setting).

Software

Emulation by software

Hardware

Hardware support is used.

Off

The open GL support is disabled (leads to minimal changes in the graphic display).

Note  

Once open GL support is reactivated, the ASCMO-Cycle Generator must be restarted.

CSV Export Cell Separator

The character that separates the table cells when exporting to CSV format.

CSV Export Decimal Separator

The decimal separator in a CSV export.

Simulink Version

This is where you select the Simulink version you want to use with the ASCMO-Cycle Generator. Available selections are the Simulink versions installed on your computer.

Reset

Resets all options to their default values.

OK

Applies your settings and closes the window.

Cancel

Discards your settings and closes the window.
