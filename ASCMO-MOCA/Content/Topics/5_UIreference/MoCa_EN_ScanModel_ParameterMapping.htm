Scan Model <model_name> for Parameter Mapping Window

The "Scan Model <model_name> for Parameter Mapping" Window contains the following elements:

Scalars, Matrix, Curves, Maps, Group Axes, Enums checkbox

These options show/hide parameters of the respective type. Enumerations will be displayed in a separate tab in the Models step.

Use Table Name checkbox

Select this option when you want to use the name of the Table Data Simulink column.

Custom Blocks

Opens a dialog where you can define patterns for recognition of axes and table values.

"Filter" field

Enter the string by which you want to filter the list, then press <Enter>.

The filter is not caps-sensitive. You cannot use wildcards.

table

  • "Create Mapping": If activated, the mapping is created when you close the window with Add Mappings or Replace Mappings.

  • "MOCA Parameter": Shows the MOCA parameter with identical name as the Simulink parameter. If no MOCA parameter with identical name is available, this column shows Create New, indicating that a new MOCA parameter will be created if you close the window with Add Mappings or Replace Mappings.

    If desired, you can use the combo box to map a different MOCA parameter to the Simulink parameter or to create a new MOCA parameter via Create New.

  • "Simulink Path": Shows the subsystem hierarchy of the Simulink parameter. If the parameter is located on the top level, this column is empty.

  • "Simulink Name": Names of the parameters found in the Simulink model.

  • "Type": Lists the type of the Simulink parameter. Possible values are Scalar, Curve, Map, Cube-3D and Cube-4D.

  • "Table Data Simulink": Shows the content of the "Table data" or "Constant value" field in the Simulink block parameters.

  • "Breakpoints 1" - "Breakpoints 4": Show the content of the "Breakpoints <n>" fields in the Simulink block parameters.

    Note  

    ASCMO-MOCA requires MATLAB variable names in the "Table Data Simulink" and "Breakpoint <n>" columns. If, e.g., a vector [1 2 3] is shown in a "Breakpoint <n>" column, the parameter cannot be used by ASCMO-MOCA.
(De)Activate All

Activates/deactivates all options in the "Create Mapping" column.

Add Mappings

Adds the selected mappings to the "Parameters Mapping" list. Existing entries in that list are kept.

Replace Mappings

Removes any existing entries in the "Parameters Mapping" list and adds the selected mappings.

Cancel

Closes the window and discards the settings.
