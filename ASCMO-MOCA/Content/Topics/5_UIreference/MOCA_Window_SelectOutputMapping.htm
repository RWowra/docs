Select <Model> Outputs

Models step > Output Mapping area > Add

The Select <Model> Outputs window for ASCET models, FMU models or TSim Plugins contains the following elements:

<Model> Output

Select one or multiple outputs to add mappings to the Output Mapping list. You can use the standard Ctrl/Shift selection functions in the table, or click and hold Lmb and drag the cursor over the cells/rows you want to select.

Import

Click to import a lab file parameter list to use an existing output mapping.

OK

Adds the selected mappings to the Output Mapping list. Existing entries in that list are kept.

?

Opens the online help in the appropriate context.

Cancel

Discards your settings and closes the window.
