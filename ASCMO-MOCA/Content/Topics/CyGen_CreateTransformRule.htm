Creating a Conversion Rule

To create and set up a conversion rule, proceed as follows.

Note  

The External Conversion RuleExternal Validation Rule template can be used only in the P-code version of ETAS ASCMO (ASCMO_SDK license).

  1. In the ASCMO-Cycle Generator window, select Conversion Rules → Configure Conversion Rule.

    The "Select a conversion rule" window opens.

  2. To create a new conversion rule, do the following.

    1. In the conversion rules list of the "Select a conversion rule" window, select a conversion rule template or an existing conversion rule.

    2. Click on Duplicate.

      A copy of the selected conversion rule or template is created and selected. It is named <original_name> (new) or <original_name> (<i>).

      The copy of an existing conversion rule contains all settings of the original, the copy of a template is empty.

  3. To set up a conversion rule, do the following.

    1. In the conversion rules list of the "Select a conversion rule" window, select an existing conversion rule.

      You cannot edit the properties of the conversion rule templates (Simple Vehicle Model and External Conversion Rule).

    2. Open the "Configuration" tab.

    3. In the "Configuration" tab, "Conversion Rule Settings" area, specify the conversion rule according to your needs.

      See the tooltips and "Configuration" tab for further information.

    4. In the "Input mapping" table, assign the cycle columns that contain the input data to the inputs.

       

      Input

      content of cycle column

      Velocity profile

      vehicle speed

      Gear profile

      the gear engaged at each time of the driving cycle

      Height profile (optional)
      the height at each time of the driving cycle

      e.g., the height above sea level, or the height relative to a particular point of the test drive

    5. In the "Output mapping" table, assign the outputs (i.e. engine speed and engine torque) to suitable cycle columns.

      If no suitable column exists, select New to create a new column.

  4. To select a conversion rule, do the following.

    1. In the conversion rules list of the "Select a conversion rule" window, select an existing conversion rule.

    2. Click on Select and close.

  5. To deselect the current conversion rule without selecting a new one, do the following.

    • In the main window of the ASCMO-Cycle Generator, select Conversion Rules → Deselect Current Conversion Rule.

The selected conversion rule is shown in the bottom bar of the main window of the ASCMO-Cycle Generator. You can use it to convert a cycle.
