Edit Training/Test Data

Data menu > Edit > Training and/or Test Data

The read training data are displayed as a table and can be changed in the table.

You can use the standard Ctrl/Shift selection functions in the table, or click and hold Lmb and drag the cursor over the cells/rows you want to select.

Use F2 or Double-click a cell to edit the value.

A distinction can be made between the display of training data (> Training Data) and - if available - test data (> Test Data).

The Edit Training/Test Data window contains the following elements:

File

  • Plot Selected Data
    Marked areas of the table (across up to 3 columns) can be displayed in simple plots.
  • Show Current Row in ISP View
    The values of this row are set in the ISP view.
  • Export...
    The content of the window is written to an Excel file (*.xls).
  • Apply Changes and Retrain
    Saves the change to the dataset and performs a new model training.
  • Mark Selected Rows in Other Views
    Marks the selected rows of the table as crosses in the intersection plots.
  • Remove Values <= 0 of All Outputs
    Some measurements contain negative values for the output variables, although the output can actually never become negative (e.g. for emissions). If not already done during the data preparation, all values less than 0 can be deleted here from the measuring data.
  • Remove Values <= 0 of Selected Outputs
    Removes values <= 0 only for selected outputs (= columns).
  • Add Calculated Data...
    This function allows adding a new column (output) with the values of existing inputs and outputs to the training dataset, which is also displayed in the intersection plots.

    Enter a name for the new output (Node) and, if necessary, its physical unit (Unit). The formula with which the output (as function of existing inputs and outputs) is calculated can be defined under Expression.
    Example: 10 * speed + 5 * Fuel_mass
    After clicking OK, a model training (simple linear model) is started for this output.
  • Close
    Closes the Edit Training Data window.
