"Model Configuration Manager" Window

The "Model Configuration Manager" window (Model > Configuration Manager) contains the "Edit and Compare" tab and the "Status and Training" tab:

"Edit and Compare" tab

The "Edit and Compare" tab contains the following elements:

Named Configurations

Search field and list

The list contains all named configurations. You can use the standard Ctrl/Shift selection functions in the table, or click and hold Lmb and drag the cursor over the cells/rows you want to select. Type in the search field to quickly find a configuration.

New

Click to add a new configuration. An new empty "Model Configuration All Outputs" window opens.

Open

First, select the desired configuration from the list, then click Open. The "Model Configuration All Outputs" window of the selected configuration opens.

Clone

First, select the desired configuration from the list, then click Clone. A new "Model Configuration All Outputs" window opens with the same settings as the selected configuration. You must assign a new name.

Rename

First, select the desired configuration from the list, then click Rename. Edit the name of the configuration in the opened window. There cannot be two identical names.

Delete

First, select the desired configuration from the list, then click Delete to delete it.

Properties

Table

The table shows the differences and similarities between the selected configurations. Differences are highlighted in red.

"Only show differences" checkbox

Activate the checkbox to display only the differences between the selected configurations in the table.

"Status and Training" tab

The "Status and Training" tab contains the following elements:

The table shows all models available for output. Configurations are shown as column headers and outputs are shown as row headers.

The combinations of configuration and output are color coded:

green: trained.

red: outdated. Means a trained model is available, but the training data has changed since training.

gray: untrained.

Activate the checkboxes of the desired configuration and output combinations. Click Train Selected to generate and train a model for the selected combinations.
