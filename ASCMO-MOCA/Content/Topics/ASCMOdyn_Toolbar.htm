Toolbar (ASCMO-DYNAMIC)

The toolbar contains a number of buttons that can be used to call up the following functions.

  • New Project

    Opens the start window of ASCMO-DYNAMIC, where you can create a new ASCMO-DYNAMIC project (by importing training data).

  • Open Project

    Opens an ASCMO-DYNAMIC project (*.ascmodyn).

  • Save

    Saves the current ASCMO-DYNAMIC project.

  • Scatter Plot: All Data

    Training data, validation data, and test data of the outputs, input gradients, input and sample no. will be generated and shown as scatter plots.

  • Phase Plot and IACF &Outputs

    Phase plots of the outputs are generated and shown. See also Phase Plot and ACF/IACF Outputs.

  • Downsampling

    Opens the "Resampling" window, where you can change the sample time of the datasets. See also Resampling.

  • Start Model Training

    Opens the "Model Training" window, where you can start the model training. See also Menu Model →Training.

  • Model Properties

    Opens the "Model Properties" window, where you can define individual model settings for each output. See also Model Configurations (ASCMO-DYNAMIC).

  • Cross Validation on Training Set

    If no test data are available, a cross validation is started (see Cross Validation on Training Set).

  • Show Model Sigma

    See Show Model Sigma.

  • Show Convex Hull on In- and Outputs

    Shows convex hulls for the data (see View Menu (ASCMO-DYNAMIC)).

  • Zoom In

    Clicking in the plot will enlarge the plot representation.

  • Zoom Out

    Clicking in the plot will reduce the plot representation.

  • Pan

    Thus, the plot can be shifted within the window.

  • Group Windows

    Groups all currently opened ASCMO-DYNAMIC windows so that they are all visible at the same time

  • Group Child Windows

    Closes all currently opened ASCMO-DYNAMIC windows except for the main window

  • Close Child Windows

    Closes all currently opened ASCMO-DYNAMIC windows except for the main window

  • Online Help

    Opens the online help (<F1>).
