"Select Axes" Window

The "Select Axes" window contains the following elements:

List

This tab allows list-based selection of plot combinations.

  • X-Axis list and Y-Axis list

    These lists contain all parameters that can be plotted, e.g. inputs and outputs of the current project.

    In each list, you can select one or more inputs to be used as X or Y axis for a 2D plot.

  • Sort List

    If activated, the list above the checkbox is sorted alphabetically.

  • Select

    Adds combinations of each selected X axis with each selected Y axis to the Show Y vs. X list. Existing combinations are added a second time.

  • Remove

    Removes the selected combinations from the Show Y vs. X list.

  • Show Y vs. X

    This list displays all selected combinations. You can use the Up and Down buttons to move a selected combination in the list.

    Each combination is plotted once, even if it appears more than once in the list.

Matrix

This tab shows a matrix with checkboxes for all combinations. The row determines the Y axis of the plot, the column determines the X axis.

If you click OK or Apply, each selected combination is plotted.

Deselect All

Deselects all combinations.

Select Minimal

Selects as many combinations as required so that each axis appears in one plot.

Select All Combinations

Selects as many combinations as required so that each axis is plotted against all other axes. The following combinations are not selected:

  • an axis against itself (e.g., speed vs. speed)
  • the inverse of a combination already selected (if, e.g., speed vs. load is selected, then load vs. speed is not selected)

Save Axes Setup

Saves the current selection of axes to an *.ini file

Load Axes Setup

Loads an existing selection of axes.

OK

Applies your settings and closes the window.

Apply

Applies your settings, but does not close the window.

Cancel

Discards your settings and closes the window.
