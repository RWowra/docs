"Generate new cycles" Window

This window opens when you select Cycle Generators → Generate Cycles, or press the Generate button in the ASCMO-Cycle Generator window.

The "Generate new cycles" window contains the following elements:

"Open configuration" field and Configure button

The Configure button opens the "Select a generator" window, where you can select and configure cycle generator.

"Cycle Information area

  • "Number of New Cycles" field

    Enter the number of cycles you want to generate.

  • "Duration" field

    Enter the cycle duration in seconds.

  • Randomize new cycles option

    Additionally distort new cycles. This option is only recommended if little training data is available.

  • "Initial values" area

    random option

    If activated, random start values are used.

    "<input>" fields

    one field for each input selected in the "Input Selection" table of the "Select cycle generator" widow

    Enter the start values for the driving cycle to be created.

    Note  

    These fields are only available if random is deactivated.

  • "Name prefix" field

    Enter the name for the generated cycle.

    If you generate more than one cycle at a time, or if a cycle with the given name already exists, the new cycles are named <cycle_name> (1), <cycle_name> (2), etc.

"Cycle Validation" area

  • "Validation Rule" combo box

    Use this box to select if you want to apply no validation rules (None), the currently selected validation rule (Selection), or all validation rules (All).

  • "Operation" combo box

    Use this box to select the way cycles that violate the selected validation rule(s) are treated.

    None

    Invalid cycles are treated like valid cycles.

    Discard invalid

    Invalid cycles are excluded from cycle generation.

    Greedy search

    The validation rules are repeatedly applied, and the results are analyzed. The cycle part before the first violation of the validation rules is kept, the remainder is sampled again. With that, a valid cycle may be reached by n iterations.

  • "Number of attempts" field

    Enter the number n of tries per generated cycle.

Generate

Generates the selected number of cycles.

Cancel

Discards your settings and closes the window.
