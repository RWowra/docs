Creating a Cycle Generator

To create, set up, select and train a cycle generator, proceed as follows.

  1. In the ASCMO-Cycle Generator window, select Cycle Generators → Configure Generator.

    The "Select a generator" window opens.

  2. To create a new cycle generator, do the following.

    1. In the generator list of the "Select a generator" window, select a generator template or an existing cycle generator.

    2. Click on Duplicate.

      A copy of the selected cycle generator or template is created and selected. It is named <original_name> (new) or <original_name> (<i>)1 <i> being an integer number .

      The copy of an existing cycle generator contains all settings of the original, the copy of a template is empty.

  3. To set up a cycle generator, do the following.

    1. In the generator list of the "Select a generator" window, select an existing cycle generator.

      You cannot edit the properties of the cycle generator templates (Markov chain and Recurrent neural net).

    2. Open the "Configuration" tab.

    3. In the "Configuration" tab, "Generator settings" area, specify the cycle generator according to your needs.

      See the tooltips and "Configuration" tab for further information.

    4. In the "Input Selection" table, "Select" column, activate the options of the columns you want to use for cycle generation.

  4. To train the cycle generator, do the following.

    1. In the generator list of the "Select a generator" window, select an existing cycle generator.

    2. Click on Train modified.

      If the cycle generator configuration has changed, training starts and the "Select a generator" window closes.

      Otherwise, the "Select a generator" window closes without starting a training.

      or

    3. Click on the arrow button next to Train modified and select Train selected from the pop-up menu.

      Training of the cycle generator starts and the "Select a generator" window closes.

    Training results are shown in the log window of the ASCMO-Cycle Generator.

  5. To select the cycle generator without training, do the following.

    1. In the generator list of the "Select a generator" window, select an existing cycle generator.

    2. Click on the arrow button next to Train modified and select Close without training from the pop-up menu.

  6. To deselect the current cycle generator without selecting a new one, do the following.

    • In the main window of the ASCMO-Cycle Generator, select Cycle Generators → Deselect Current Generator.

The selected cycle generator is shown in the bottom bar of the main window of the ASCMO-Cycle Generator. You can use it to generate new cycles.
