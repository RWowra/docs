Exporting a System

Export System feature exports a system along with its artifacts in a *.cssys format.

The elements below are considered while exporting a system:

  • The model instances in a system
  • Models in a target which is associated with a system
  • Selected parameters of the model instances
  • Connections in a system
  • OS Configuration
  • The System Settings made for a project
  • The Target Settings made for a project
  • The build artifacts such as the generated code and the experiment which is optional

To export a system

  • Right-click on a system in the project panel.
  • A context menu is displayed.
  • Click .
  • The "Export System(s)" dialog box is displayed.

Fig. 92: The "Export System(s)" dialog box

  • Specify the location to export a system or click Browse to select it.
  • Check the checkbox "Pack build artifacts" to export the build artifacts.
  • Click Export.
  • The "Replace or skip export system(s)" dialog box is displayed if the selected system(s) is(are) already available in the destination.
  • Click either "Replace the system(s) in the destination" to replace the existing system(s) or "Skip these system(s)" to ignore the system(s).
  • A system is exported as *.cssys format along with its artifacts.

Exporting the Multiple Systems

Export System feature also helps to export two or more systems. You can select two or more systems by holding <Ctrl> key and the systems.
