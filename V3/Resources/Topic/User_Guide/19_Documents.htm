Technical Documents

COSYM provides you the important information on how to install and use the product which are described in the following documents listed below:

Getting Started Guide

Assists you in installing COSYM. It provides information on hardware and software requirements, license information and many more information.

Available with the provided COSYM Installation path or the DVD. It is also available at Start menu → All Programs → ETAS COSYM → Manuals and Tutorials → COSYM_V3.4.1_Installation Guide.pdf or go to Help menu → Documentation in the project screen and project dashboard and click COSYM_V3.4.1_Getting_Started_Guide.pdf.

Release Notes

It details about Licensing, Prerequisites, the changes or enhancement made to COSYM V3.4.1, already reported known issues, and a link to known issue report.

It is available at Start menu → All Programs → ETAS COSYM→ Manuals and Tutorials → COSYM_V3.4.1_Release_Notes.pdf.

Software Compatibility List

A list of other applications which are compatible with COSYM.

It is available at Start menu → All Programs → ETAS COSYM→ Manuals and Tutorials → COSYM_V3.4.1_Software_Compatibility_List.pdf.

Online Help

Provides the complete information on how to use the product. A tutorial is included in the help document which describes to show how simulation is conducted in RT simulator and SiL use case using COSYM.

The document is embedded in the COSYM Product. It can be viewed by clicking ? icon in the project screen or go to Help menu and click Help Manual.

User Guide

It also provides the complete information on how to use the product in the pdf format.

It is available at Start menu → All Programs → ETAS COSYM→ Manuals and Tutorials → COSYM_V3.4.1_User_Guide.pdf.

Tutorial Guide

A tutorial is included in the manual which describes to show how simulation is conducted in RT simulator and SiL use case using COSYM.

It is available at Start menu → All Programs → ETAS COSYM→ Manuals and Tutorials → COSYM_V3.4.1_Tutorials.pdf.

VNET Guide

This document provides the information on VNET and describes how to access VNET through APIs.

It is available at Start menu → All Programs→ ETAS COSYM→ Manuals and Tutorials → COSYM_V3.4.1_vNET_guide.pdf.

ETAS Experiment Environment - an Overview

This document provides an overview on ETAS Experiment Environment.

It is available at Startmenu → All Programs → ETAS COSYM → Manuals and Tutorials → ETAS_Experiment_Environment.pdf.

SiL Python API Guide

This document provides how to access TargetClient API from Python for the SiL ATS use case.

It is available at Startmenu → All Programs → ETAS COSYM → SiL_Python_API_Guide.pdf.

This document provides the information on how to create the real-time plugins, integrate them into a COSYM project and work with them in the ETAS EE.

It is available at Start menu → All Programs → ETAS COSYM → Manuals and Tutorials → RT_Plugins_User_Guide.pdf.

Network Model User Guide

Provides the information on how to create a Network restbus model and assist you on working with Network restbus model.

It is available at Start menu → All Programs → ETAS COSYM → Manuals and Tutorials → Editors → Network_Model_User_Guide.pdf.

ASAM-XIL Documentation

The XIL API, created by ETAS, is a programmable interface for remote controlling of COSYM.

Note 

COSYM is not fully ASAM XIL compliant. ASAM XIL 2.1.0 is the current version which is supported by COSYM.

The ASAM-XIL document is available in the <COSYM Installation Directory>\ASAMXil\Manual\ETAS_XILAPI.chm file.
