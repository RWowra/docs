Deployment Management

Deployment Management allows to create the targets. The default targets available in the Deployment Management is SiL_target. Either you can use the default target or you can create a new target.

The targets can be deleted in the Deployment Management, if it is not mapped to any of the systems. The default targets can also be deleted if they are not mapped.

Refer to the sections listed below on how to perform these operations:

  • Creating a Target
  • Target
  • Virtual Bus
  • Deleting a Target

To open the "Deployment Management", go to View menu and click Deployment Management.

Fig. 157: The "Deployment Management" window

Note 

The GUI is not available for the SiL Linux target. The operations such as creation, modification, configuration can be performed only through COSYM REST APIs.
