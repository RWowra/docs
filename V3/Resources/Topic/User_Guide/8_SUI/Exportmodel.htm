Exporting a Model

Export model feature exports the model along with the COSYM artifacts (the configurations made into a model) in a *.csm format. You can exports it into its original format also. These exported COSYM artifacts can be imported into different COSYM projects to use the same configurations.

To export a model

  • Right-click on a model in the library.
  • A context menu is displayed.
  • Click .
  • The "Export model" dialog box is displayed.

The "Export model" dialog box

  • Specify the location to export the model or click Browse to select it.
  • Click Export.
  • The "Replace or skip export model(s)" dialog box is displayed if the model(s) in the selected file is already available in the destination.
  • Click either "Replace the model(s) in the destination" to replace the existing model(s) or "Skip these model(s)" to ignore it.
  • The model is exported as *.csm format and its original format (if the checkbox is selected) into the specified location.

Supporting of Multiple Model Export

The "Export Model" feature also helps to export two or more models along with the COSYM artifacts. You can select two or more models by holding <Ctrl> key in the library or a target in the Deployment Management.
