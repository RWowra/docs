Icons

The various icons or buttons are used in COSYM which helps for easy operation. Those icons are shown and described in the table below:

Icons Name in COSYM Description
New Helps to create a new project, system or C-Code, conversion, CAN restbus, network restbus model
Open Helps to open a project
Help Opens the COSYM help manual
Up-down arrow Helps to move up or move down the projects or connections
Maximize Maximizes the screen
– Minimize Minimizes the screen
Close Closes COSYM or a COSYM project
Build Starts the build process forcefully and generates a code.
Build and Run Generates and builds a code. Then it starts the ETAS EE to run simulation experiment and view the results
Run Starts the ETAS EE to run simulation experiment and view the results
Save or Save as Saves the modified changes or saves the project with a new name
Open recent Helps to open the recently opened project(s)
Import Helps to import a model
Close project Closes the currently opened project
Exit COSYM Exits the COSYM 
Properties Helps to update the project properties
Run experiment Opens ETAS EE to visualize the results
Settings Allows to make the settings in different levels in COSYM.
Toggle project panel Shows/hides the project panel in the project dashboard
Reset filters Resets the applied filters in the Connection view or Deployment view
Deployment Management Displays the Deployment Management
Create Target Creates a new target
Create connection Creates a connection or opens the auto connect window
Edit a target/task Opens the task
About COSYM Displays the information about COSYM
API Helps to open the Swagger editor to work with the COSYM REST APIs and generates client code
Contact support Displays the ETAS hotline information
Safety Advice Displays the Safety Advice document
License manager Opens ETAS License Manager
Problem report Drafts an e-mail to report a problem with the zipped log file attached to the concerned person
Expand All

Expands all the element(s) and its(their) child element(s)
Collapse All Collapses all the expanded element(s)
Change target Changes/maps a target to a system
Expand Expands an element such as model/folder/system/target
Collapse Collapses an element such as model/folder/system/target
Search Helps to search an element(s) such as model/folder/system/target/inport name/outport name
Folder A folder
FMU/C-Code An FMU/C-Code model
LMD/VXIP/Conversion model An LMD/VXIP/Conversion model
Simulink/MATLAB model A Simulink®/MATLAB® model
Hardware model A hardware model
CAN restbus model A CAN restbus model
Network restbus model A network restbus model
System A system
Target A target
Delete Deletes an element such as a model/folder/system/target/connection/task/hook
Show instances in system Show the model instance(s) in a system
Copy Copies a folder or a model
Paste Paste a copied item
Create Creates a model or a folder, adds a model to a system
Layout the graphical composer Arranges the models and their connections in an organized manner in a graphical composer
Inport of a a model An inport of a model
Outport of a model An outport of a model
Calibration variable/Measurement variable A calibration variable or measurement variable
Zoom out Zooms out
Reset to original Sets the zoom level to 100%
Zoom in Zooms in
Search Searches the connections in the Connection viewer
Show connection editor Shows the connection editor in the Connections view
Hide connection editor Hides the connection editor in the Connections view
Import Helps to import the connections/configurations from the selected file
Export Helps to export the connections/configurations into a specified file
Convert to C-Code model Converts to LCO C-Code model to COSYM C-Code model
Create Creates a category or a workpackage
Expand task Expands a task in the Deployment view
Collapse task Collapses a task in the Deployment view
Add task Adds a new task
Auto mapping Configures the OS by mapping the processes into the relevant tasks
Incremental Auto mapping Helps to apply the auto mapping feature for the unmapped processes in the model instances
Search Searches a task or an element in the task panel
Add Adds an artifacts/inports/outports/measurement variables/calibration variables/external files/macro definitions
Global Filter Filters mentioned the files in the External Files view
Message - INFO Displays all the run-time events in the Messages window
Message - WARNING Displays all the warning messages or cautions in the Messages window
Message - ERROR Displays all the errors in the Messages window
Browse Helps to browse a folder or a file
Reload simulink models Reloads the simulink models
