Run

The 'Run' feature mainly helps to open the ETAS EE to view the simulation result for the previously built system. COSYM opens ETAS EE to view the result for a system which was built earlier and there is no change made in a system after the code was generated.

Run ETAS EE

To start ETAS EE to run a simulation experiment

  • Go to Project menu and click Run.
    A window is displayed to select a system to run it.
  • Click Run.
  •  <Or>
  •  Right-click on a system.
    A context menu is displayed.
  • Click Build â†’ Run.
  • Opens the ETAS EE for the previously built system.
    If the selected system is not built earlier, then a message is shown to build it.

Refer to About ETAS EE for more details.

If the changes made into a system after the build process or the codegen folder is not available for any reason, then the message is shown in the "Select System" window to build the code again. Refer to Unmapped Processes in a System.
