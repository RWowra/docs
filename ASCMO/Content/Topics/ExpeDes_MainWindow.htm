Main Window of ASCMO-STATIC ExpeDes

The main window of ASCMO-STATIC ExpeDes features the following entries:

The main menu
  • File menu
  • View menu
  • Parameter/Model menu
  • Plugins menu
  • Help menu
The Toolbar
  • New Project
    Creates a new project.
  • Open Project
    Opens a previously saved project.
  • Save
    Saves the current project.

  • Apply changes and open Scatter Plot

    Calculates the modified experimental design new and presents it in a Scatter plot (View → Scatter Plot).

  • Apply changes and update viewer
    Updates the representation in the Viewer windows when changes are made ​​to the project.
  • Revert changes in current step
    Changes will be redeemed in the currently open window.
  • Zoom In* and Zoom Out*
    Scales the plot up or down.
  • Pan*
    The plot can be shifted within the window.
  • Rotate 3D*
    The 3D plot can be rotated in all three spatial directions.

The List of working steps

The current working window

In the main part of the window, functions of the currently selected work step are shown.

The Log window

In this area, status and error messages and tips are issued.
