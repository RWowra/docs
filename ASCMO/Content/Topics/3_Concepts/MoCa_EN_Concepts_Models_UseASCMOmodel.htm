Using an ASCMO-STATIC/ASCMO-DYNAMIC Model

Before you can use an ASCMO-STATIC/ASCMO-DYNAMIC model for the optimization, you need to map all ASCMO inputs to MOCA elements to make the model available in the function. Proceed as follows:

  1. In the model list of the Models pane, select an ASCMO-STATIC/ASCMO-DYNAMIC model.

    The lower part of the Models pane shows the ASCMO Model variant.

  2. In the mapping table, map all unmapped ASCMO inputs.

    1. Select an ASCMO input.

    2. In the "MOCA Name" column next to the selected input, open the combo box.

      The combo box offers all MOCA data channels and function nodes for selection.

    3. Select the channel or node you want to map to the ASCMO input.

    Note  

    All ASCMO inputs must be mapped to MOCA elements before you can use the ASCMO model.

  3. In the navigation, click on Function.

    The Function pane opens.

  4. Add a new node or edit an existing node.

    The ASCMO models are available in the "Nodes" area.

  5. Insert an ASCMO model in the expression.

    In addition, the name <ASCMO model name>_mdl is entered in the "Node Name" field.

  6. Click on OK.

    The node is added or changed according to your settings. In addition, a node <ASCMO model name>_mdl.<output name> is created for each model output.
