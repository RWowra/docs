Import Bounds Only

Parameters step > Import > Bounds > From File/From Open Projects

Or

Optimization step > > Import > Bounds > From File/From Open Projects

  1. Do one of the following:

    • Open the Parameters step and go to the Parameters tab.

    • Open the Optimization step and go to the Optimize tab.

  2. Do one of the following:

    • In the Parameters step, click on the button next to the Import button to select Bounds from the drop-down menu.

    • In the Optimization step, click on the button and select Import > Bounds.

  3. For both options choose whether From File or From Open Projects.

     

    Note  

    Data types Columns in CSV/Excel and Matrix in CSV/Excel are MOCA specific formats.

  4. Select the project/file and parameterset you want to import from.

    The Import Bounds window opens. It lists all parameters in the project and allows the selection of suitable bounds from the import file/project. Bounds with matching names and types are preselected.

  5. In this window, select the upper and lower bounds you want to import.

    Note  

    All parameters are displayed. If the import file contains bounds and parameters, the parameters appear in the bound selection drop-down lists.

  6. Click Import.

    The selected parameter bounds are imported and assigned to the corresponding parameters.

    If a bound from the import file/project already exists, its bounds will be overwritten, but its values will remain unchanged.
