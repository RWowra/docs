Step 7: Additional Points

In this step, repetition points can be defined which, in addition to the points of the experiment plan, should be approached repeatedly according to specific criteria and measured, if necessary.

Adding and deleting repetition points

Click Add to add a new repetition point to the list.

To remove a repetition point, select the ID of the point and click Delete.

List of Repetition Points

A repetition point is defined by the following parameters:

  • ID

    The unique identifier of the repetition point.

  • Repetition Type

    Defines the condition to be met after which the repetition point is approached in each case:

    • Every x Measurements

      The repetition point is approached after x measurements. If this option is selected, a dialog opens for entering x.

    • Overall x Repetition Points

      With this option, the repetition point is approached x times during the measurement of the entire experiment plan.

    • Include Center Points

      See Center Points.

    • Before Change of Cluster Point

      The repetition point is approached every time before the cluster point is changed.

      Note  

      This option is meaningful only if the cluster points have been sorted accordingly in the "Sorting Rules" step.

    • Before Change of ...

      A different variable of the measurement plan can be selected here that requires approaching the repetition point in each case before it is changed. The previous note about sorting also applies to this case.

    • Before New Block

      The repetition point is approached before the measurement of a new block is started.

  • Type <Input> and Value <Input>

    The repetition point is defined here.

Center Points

For each speed/load combination, a safe setting of inputs, the so-called center points, exists. At this point, the motor runs stable. The center points can be provided in the form of maps, one for each output, over the speed-load plane (or the plane of your clustered inputs).

The center points are added to the measurement plan via the Include Center Points repetition point. Set the values in the row as follows:
