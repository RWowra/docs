Importing Driving Cycle Data

To import driving cycle data, proceed as follows.

Note  

The Calibration menu is only available if operating point axes have been selected. You can set the operating point axes via the In/Outputs → Set Operating Point Axes menu option (see Set Operating Point Axes).

  1. Select Calibration → Driving Cycles

    The Driving Cycle Manager window opens.

  2. In the "Driving Cycle Manager" window, select File → Import Driving Cycle.

    A file selection window opens.

  3. In the file selection window, select the driving cycle file you want to import and click Open.

    The "Import Data" window opens. It shows as many operating points as have been defined via the In/Outputs → Set Operating Point Axes menu option. For each operating point axis in the project, one combo box with the same name as the operating point axis is available.

    ASCMO-STATIC tries to match the input names of the loaded file to the operating point axes.

  4. If the names in the project ("Project Import" column) differ from the names in the file (drop-downs in "Import Name" column), assign them manually.
  5. Set Time Base and Resampling settings.

  6. Select a "Modification Mode" (Add or Replace).

  7. If you want to use another name than the imported file, enter a name for the driving cycle.

    This name is used in the ASCMO-STATIC user interface.

  8. Click Import to start the import procedure.

  9. The driving cycle data file is imported. Once the import is finished, the "Driving Cycle Manager" window shows the driving cycle data in a table view.

    Note  

    If you want to use several driving cycles in your project, import them by consecutively using File > Import Driving Cycle and "Modification Mode" Add or by loading an already existing database (*.cycles) using File > Load Database. Then ASCMO-STATIC can use all driving cycle traces for the cycle forecast.

The imported data can be used for global optimization based on Calculation Rules which use these driving cycle data (Calibration > Prognosis > Calculation Rules: Driving Cycle Based Tab).
