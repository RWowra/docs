Main Window – ASCMO-Calibration Data Editor

The main window of the ASCMO-Calibration Data Editor contains the following elements:

toolbar
Zoom In Clicking in the plot will enlarge the plot representation.
Zoom Out Clicking in the plot will reduce the plot representation.
Pan Thus, the plot can be shifted within the window.
Rotate 3D Thus the 3D plot can be rotated in all three spatial directions.
button bar

Import

Imports parameters and their values from a file. Available import formats are DCM Files (*.dcm), comma-separated values in columns or matrix (*.csv), and Excel files with columns or matrix (*.xls, *.xlsx).

Create

Opens the "Create Parameter" window where you can create a new parameter.

See also Creating/Editing a Parameter.

Edit

Opens the "Edit Parameter" window where you can edit the selected parameter.

See also Creating/Editing a Parameter.

Duplicate

Creates a copy of the selected parameter and names it <parameter>(<n>).

Selecting two or more parameters is possible.

Delete

Deletes the selected parameter(s).

Export

Exports the parameters and their current values to a file.

Available export formats are DCM Files (*.dcm), comma-separated values (*.csv), calibration data files (*.cdfx) and m-scripts (*.m).

Optimize Grid

Opens the "Optimize Grid" window where you can optimize the grid of a map/curve.

Note  

This button can be used only for curves and maps.

Convert / Invert Axes

Opens the "Axes Convert/Invert" window where you can convert/invert parameters. See also "Axes Convert/Invert" Window.
parameters list

Besides the parameter names, the list contains the following columns:

Lower Limit

Show the lower and upper limit of the parameter values.

If desired, you can edit the limits in the parameters list.

Upper Limit

parameter table

Shows the content of the selected parameter. The look of the table depends on the parameter type.

For 3D and 4D cube parameters, the third and fourth axes are shown as separate lists.

plot area

Shows a plot of the selected parameter. The look of the plot depends on the parameter type.
