Options (ASCMO-STATIC)

File menu > Options

The Options window contains the following elements:

Help Language

This is where the language of the online help can be defined.

Advanced Settings

This activates the display of further settings (see List of Advanced Settings functions).

Note  

We recommend to restart ETAS ASCMO after activating, as the menu entries are not placed correctly immediately.

Open GL

Setting for open GL support. Possible selections:

Default (Hardware Basic)

If available, hardware support is used (default setting).

Software

Emulation by software.

Hardware

Hardware support is used.

CSV Export Cell Separator

The character that separates the table cells when exporting to CSV format.

CSV Export Decimal Separator

The decimal separator in a CSV export.

Simulink Version

This is where you select the Simulink version you want to use with ASCMO-STATIC/ASCMO-DYNAMIC. Available selections are the Simulink versions installed on your computer.

Use Full Desktop Matlab

If activated, a full MATLAB® with desktop is started, instead of a command line simulink (external). This takes more time but is helpful for debugging Simulink® models.

Autosave Training

If activated, a backup copy is saved automatically (project.autosave.ascmo):

  • after Model training
  • during Automatic Feature Selection (ASCMO-DYNAMIC)
  • during Automated Machine Learning

Reset

Resets all options to their default values.

OK

Applies your settings and closes the window.

Cancel

Discards your settings and closes the window.


List of Advanced Settings functions

    The Advanced Settings option activates, e.g., the following functions:

    ASCMO-STATIC

    • Menu item In/Outputs > Link Inputs
    • Menu item Model > Valid Model Range
    • Menu items Model > Reference Model> * > *
    • Menu item Model > Error (Leave One Out) > Show in ISP
    • Menu item Model > Model Compression> *
    • Menu items Model > Scatter Plot Valid Model Range > *
    • Entries in the context menu of the intersection plots
    • Dialog Advanced Parameter for Outputs/Model
    • Number of Repetitions field in the LOO-Error on Training Data over Training Size window

    Dialog Measured vs. Predicted / Error vs. Predicted (Leave-One-Out / Training Data / Test Data)

    • Menu item Extras > Set Outlier Threshold

    Dialog Single Result Optimization

    • Dialog Settings > Edit > Advanced Parameters

    Dialog Multi Result Optimization

    • Dialog Settings > Edit> Advanced Parameters

    Dialog Global Optimization

    • Dialog Settings > Edit > Advanced Parameters
