Starting ASCMO-DYNAMIC ExpeDes

ASCMO-DYNAMIC ExpeDes can be started from the "ASCMO Desk" window or from the Windows start menu. Proceed as follows:

  1. Do one of the following:

    • In the ASCMO-DESK window, click on Transient Test Planning.

    • In the Windows Start menu, open the ETAS ASCMOV5.14 program group and select ASCMO ExpeDes Dynamic V5.14.

    The start window of ASCMO-DYNAMIC ExpeDes opens.

  2. To open a new project in ASCMO-DYNAMIC ExpeDes, click on Start ExpeDes.

    The main window of ASCMO-DYNAMIC ExpeDes opens with some default settings.

  3. To open an existing project in ASCMO-DYNAMIC ExpeDes, do the following:

    1. In the ASCMO-DYNAMIC ExpeDes start window, click on Open ExpeDes Dynamic Project.

      A file selection window opens.

    2. In the file selection window, select the *.exde file you want to open, then click on Open.

      The main window of ASCMO-DYNAMIC ExpeDes opens. The selected project is shown.

  4. To open the demo project installed with ASCMO-DYNAMIC ExpeDes, click on the Open ExpeDes Dynamic Demo Project link below the buttons.
