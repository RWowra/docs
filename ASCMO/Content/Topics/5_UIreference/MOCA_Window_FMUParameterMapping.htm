Add FMU to MOCA Parameter Mapping

Models step > Parameter Mapping area > Add

The Add FMU to MOCA Parameter Mapping window contains the following elements:

MOCA Parameter

Shows available parameter, where you can select one out of the list. You can also <Create New Parameter>.

Type

Shows the type of selected parameter. If you create a new parameter, you can define the type by selecting out of the list.

Name

Shows parameter name. If you create a new parameter, you can enter a name in the text field.

Add FMU Field

Shows available FMU elements for mapping to a MOCA parameter.
You can minimize the parameter list by entering a name into the respective input field.

Breakpoints

Shows available breakpoints for mapping. This must fit together, e.g. the interpolation points must match the map/curve.

Maps: needs a field and 2 breakpoints axis, in addition you can define the conversion as a transpose field.

Curve: needs a field and one breakpoint

New parameter (with Cube-4D type): needs a field and 4 breakpoints.

Transpose Table Data

Select Transpose Table Data for Maps or Matrix (Parameter Type).

Column Major

Usually the data is interpreted in row-major order. If data should be interpreted in column-major order, select the checkbox.

Autodetect

Autodetect matching MOCA parameter, type and related axes from patterns.

Patterns

Opens the Patterns for Curve/Map mapping, where you can define string patterns to detecting Curves or Maps.

OK

Maps the selected elements as parameters with the given name, if you entered one and the set field. Closes the window.

Apply

Applies your settings, but does not close the window.

Cancel

Discards your settings and closes the window.
