Scan <Model Name> for Parameter Mapping

Models step > Parameter Mapping area > Scan

The Scan <Model Name> for Parameter Mapping window contains the following elements:

Scalars, Matrix, Curves, Maps

These checkboxes show/hide parameters of the respective type.

Patterns

Opens the Parameters Mapping window where you can define string patterns for mapping of variables to Curves/Maps.

Scan Simulink

Open Simulink Model to scan for lookup-Tables and find variables of Curve/Map objects in the FMU. This is useful, if string pattern matching does not find Curve/Map objects.

Filter

Enter the string by which you want to filter the list, then press <Enter>.

The filter is not caps-sensitive. You cannot use wildcards.

Only parameters whose FMU names contain the search string are displayed. If you enter the search term Ignition, for example, all rows containing "ignition" are shown.

Shows/hides search list for a quicker selection of MOCA Parameter. Select the MOCA Parameter drop-down list within the table first to browse through list.

Table

  • Create Mapping

    If activated, the mapping is created when you close the window with Add Mappings or Replace Mappings.

  • MOCA Parameter

    Shows the MOCA parameter with similar name as the FMU parameter. If no MOCA parameter with similar name is available, this column shows Create New, indicating that a new MOCA parameter will be created if you close the window with Add Mappings or Replace Mappings.

    If desired, you can use the drop-down list to map a different MOCA parameter to the FMU parameter or to create a new MOCA parameter via <Create New>.

  • Type

    Shows the type of selected parameter. If you create a new parameter, the type is derived by FMU parameter.

  • Table Data FMU

    Shows the content of the Table Data/field of the parameter.

  • Breakpoints 1 - Breakpoints 4

    Shows the name of the FMU parameter.

(De)Activate Shown

Activates/deactivates all checkboxes in the Create Mapping column.

?

Opens the context sensitive Help (<F1>).

Add Mappings

Adds the selected mappings to the Parameter Mapping list. Existing entries in that list are kept.

Replace Mappings

Removes any existing entries in the Parameter Mapping list and adds the selected mappings.

Cancel

Discards your settings and closes the window.
