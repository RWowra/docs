Importing Measured Dynamic Data

To import measured dynamic data, proceed as follows.

  1. In the ASCMO-DYNAMIC start window click New in the menu panel on the left.

  2. Click (Import Dynamic Data).

    The ASCMO Data Import window and a file selection dialog open.

  3. In the file selection dialog, enter or select the path and name of the file you want to import.

    You can import Excel (*.xls, *.xlsx) and MDA export (*.ascii) files, lists of comma-separated values (*.csv, *.txt) and MDF files (*.dat, *.mf4, *.mdf, *.mf3).

  4. Click Open.

    If the file contains several work sheets, the Select Sheet window opens.

  5. In that window, select the desired work sheets and click OK. You can use the standard Ctrl/Shift selection functions in the table, or click and hold Lmb and drag the cursor over the cells/rows you want to select.

    The Time Vector / Sample Time window opens.

  6. From the Time Vector drop-down list, select the column that contains the time.

    or

    Type the desired sample time in seconds in the Sample Time input field.

    The content of the sheets is loaded into the ASCMO Data Import window. The data as well as inputs and outputs are displayed.

    Note  

    If you have selected multiple sheets, note the following:

    The first selected sheet is used as reference for the channel names. If another selected sheet does not contain all the channel names of the first sheet, that other sheet is excluded from the import and a warning is issued in the log window.

  7. Assign the available data to inputs and outputs.

  8. Optionally:

    • Display and check the data.

    • Check the relevance of the inputs.

    • Set Time Base Options.

    • Change the Dataset Name in the input field.

  9. In the Dataset Labels field assign the desired labels to the dataset:

    The Training label is assigned by default. Models can be trained on labels in ASCMO-DYNAMIC, allowing you to combine multiple datasets.

    1. Use the x on the label or Del/Backspace to remove the label.

    2. Type into the field to select an existing label from the list or press Enter to create a new label.

      You can edit the labels later in the Manage Datasets window.

  10. Click OK to import the data into ASCMO-DYNAMIC.
