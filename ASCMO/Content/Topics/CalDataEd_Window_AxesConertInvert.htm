"Axes Convert/Invert" Window

This window is opened with the Convert / Invert Axes button in the ASCMO-Calibration Data Editor.

The look of the "Axes Convert/Invert" window depends on the selected parameter type.

The following window elements are always visible:

  • "Type" combo box and formula field

    Use the drop-down list to select the parameter type of which you want to convert/invert the axes. Available selection: Curve, Map, Cube-3D, Cube-4D

    The formula field shows the appropriate formula, from y = f(x) for curves to z3 = f(x, y, z, z2) for 4D cubes.

  • parameter table

    This table contains all parameters of the selected type.

    Column name

    meaning

    "Apply To"

    For each parameter you want to convert/invert, activate the combo box in this column.

    "Parameter"

    This column lists all parameters of the selected type

    "Monotone in * = x or y or z or z2"

    Indicates if the parameters are monotone along the axis *.

    Note  

    The number of these columns corresponds to the number of parameter dimensions.

  • Create

    Performs the conversion/inversion of each selected parameter and creates new parameters for the results.

  • Close

    Closes the window and discards your settings.

If at least one combo box in the "Apply To" column is activated, the remaining window elements are visible.

  • "Result Axes" area

    Use the drop-down list to convert/invert the axes of the parameters. Invert a curve, for example, by exchanging the x and y axes.

  • "Fitting Smoothness" field

    Enter a smoothing factor for the resulting parameter(s). A higher value means stronger smoothing.

  • "Result Support Vectors" area

    Use the drop-down list to select a specification mode for each resulting axis.

    Optimized 

    The axis points are distributed in a way that the resulting axis is optimally covered.

    Begin/End

    The input fields are used to specify lower and upper bound of the axis.

    The axis points are evenly distributed.

    Support Vector

    The axis points are specified as a vector, e.g., [13.14 31.44 49.73 68.03 86.33 ], in the input field.

    This mode can be used to specify unevenly distributed axis points. If you add or delete an axis point, the value in the respective "Count" field is updated.

    From Data

    Use the axis points from the input parameter.

    The "Count" fields are used to determine the number of axis points. If you change the value while Support Vector is selected in the respective combo box, a new vector with the new number of equidistant axis points is created.

  • "Suffix" field

    The string entered here is added to the parameter name.

    If a parameter named <parameter_name>_<suffix> already exists, you are asked if you want to overwrite that parameter when you click on Create.
