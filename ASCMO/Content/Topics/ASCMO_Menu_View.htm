View Menu (ASCMO-STATIC)

The View menu consists of the following entries:

Save as Bitmap

Saves a screenshot of the current ISP view (PNG, BMP or JPG).

Copy to Clipboard

Stores the current ISP view for further use in the clipboard.

Show Model Sigma

Enables the display of the standard deviation in the intersection plots. The dashed red lines show the size of a standard deviation.

Sigma indicates the confidence area of the model prediction for the selected settings – the mean of all remeasurements falls within this confidence area with a probability of 68.3% (a standard deviation).

Show Valid Model Range

Enables the display of the model validity area in the intersection plots. The gray lines indicate the areas in which the model is not valid.

Show Hull On Inputs (green)

Enables the display of the hull in the intersection plots (green dashed line).

This hull provides another indication concerning the model validity. The hull is the envelope of the measuring points if all variables are plotted against all of them.

Note  

Since no multi-dimensional limits can be recognized here, the hull does not exactly correspond to the envelope curve of all data in all dimensions.

For certain model types, there is no display of the "Valid Model Range", so that the hull remains the only statement about the model validity.

Show Inputs Bounds/Map Bounds over OP (yellow)

Limits the area between minimum and maximum of the input data (yellow dashed line).

Show Marked Data

If this option is enabled, data selected in plot windows are marked as colored crosses in the ISP view.

Show Raw Data for all Outputs

Displays the training data in the intersection plots (gray dots).

Set Zoom

Opens a dialog window in which minimum height (in pixel) of the individual intersection plots can be defined. This value is also retained if the ASCMO-STATIC main window is reduced in size.

Show Grid

Enables the display of grid lines in the intersection plots.

Group Windows

Groups all currently opened ASCMO-STATIC windows so that they are all visible at the same time.

Group Child Windows

Groups all currently opened ASCMO-STATIC windows so that they are all visible at the same time except for the main window.

Close Child Windows

Closes all currently opened ASCMO-STATIC windows except for the main window.
