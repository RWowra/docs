Ports View

The "Ports" view displays all the available inports and outports of a model or model instance in the model dashboard.

The properties, behavior, and the operations of "Ports" view differs from C-Code model and conversion models. These are explained in the individual section of C-Code (Ports of C-Code Model) and conversion model (Ports of Conversion Model) respectively.

To view the Ports view

  • Open a model dashboard (To open a model dashboard for a model/model instance).
  • Click Ports.
  • The fields in the "Ports" view are displayed.

Fig. 116: Ports tab in a model

Fields of Ports view

Listed below are the fields which are available in the ports view.

  • Name
    Name of an port.
  • Type
    Displays the data type of an inport such as real, string, boolean, and integer.

Page navigation helps you to navigate the pages. By default, 25 rows can be viewed in a page. It allows you to scroll-up or scroll-down in a page. The number of rows can be changed to 50, 100, or 500 rows to display in a page.

You can search the ports by typing the port name in the field.

Adapting Initial Values in Model Instances

In the model instances, you can assign the initial values which will be used in the simulation. It helps to accomplish an initialized system before the simulation starts. The assigned values will overwrite the default values during initialization phase of simulation. You can view the "Initial value" column in the model instances only and you can edit the initial values for all data types. The "Name" and "Type" fields are non-editable in the model instances.

Fig. 117: Ports tab in a model instance with "Initial value" column

When a new model is created in COSYM and added to a system, the default value for a port is assigned as shown in the Tab. 19.

When a model is imported into COSYM, the initial value is read and inherited if it is available in the source model. Else, it is assigned as shown in the Tab. 19.

To edit the initial values of a port

  • Double-click on any port.
  • The "Port Details" dialog box is displayed as shown in the .

The "Port Details" dialog box

  • Update the value in the "Initial value" field.
  • Click Update.

Note 

If you have defined an initial value for a port in an instance and later you change the data type of that port in a model, then the initial value of the corresponding port is changed to the default value.

Refer to Initial Value for the limitation of initial values.
