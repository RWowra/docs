Calibrating Values

Calibration can be done for Parameters (Calibration values or Characteristic elements of models) with arrays, maps, curves, and other data types in INCA for all FMUs and Simulink imported models. When you simulate a system (or simulation is started), the changes will be reflected in EE and INCA as well. If the Calibration variable is changed from ETAS EE, then INCA user can upload the data again to see the changes.

For more information, refer to "INCA User Manual for calibration".

Measuring (Reading and Illustrating Measurements or Model Outports)

The measurement task consists of sampling all inports, outports and measurement values of model over a certain period of time. It is also possible to record the sampled values in a file and save it to disk.

Using INCA, you can analyze the data in a histogram or function view (Oscilloscope). The rate of reading data can be changed based on "Events" which are already defined in the "Properties" view in COSYM with assigned "Tasks".

To start measuring in INCA

  • Start the simulation in ETAS EE.
  • Select Measurement → Start Visualization.
  • When using this feature routinely, it is more convenient to use the accelerator key code <F11> to start a measurement.

To stop measuring

  • Select Measurement → Stop Measuring.
  • When using this feature routinely, it is more convenient to use the accelerator key code <F9> to stop a measurement.

An Example to Add a Measurement in YT Oscilloscope in INCA

The measurement variables can be assigned to different events. You can read the specific rate of data which is assigned to the measurement variables in COSYM.

You can select a measurement variable, start the simulation and visualize the result.

To read a variable

  • Right-click on a measurement variable which you want to read.
  • Click Add to → Layer_1 (Active) → New → YT Oscilloscope (V7.1).
