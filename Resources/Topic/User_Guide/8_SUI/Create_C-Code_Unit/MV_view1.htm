Measurement Variables of C-Code Model

The "Measurement variables" view in a C-Code model displays all the available measurement variables of a model/model instance and helps to add the measurement variables also.

The following fields are shown in the "Measurement Variables" view.

  • Name
    Name of a measurement variable.
  • Type
    Displays the data type of a measurement variable. It supports the data types such as boolean, enum, real32, real64, sint8, sint16, sint32, uint8, uint16, and uint32. The real64 data type is selected by default.

To update a measurement variable

  • Double-click on any measurement variable.
  • A "Measurement Variable Details" dialog box is displayed as shown in the Fig. 125. By default, a variable is created with the real64 type.

Fig. 125: The "Measurement Variable Details" dialog box

  • You can rename the "Name" of the measurement variable.
  • Select the data type from the "Type" drop-down menu.
    If enum data type is selected, you can assign the values as described inTo change the ENUM data type in Measurement Variable.
  • Click Update to save the changes.

To change the ENUM data type in Measurement Variable

When you select the enum data type from the drop-down menu, the dialog box is updated with the "Value" field where you can assign the required value(s) in the '#' and 'Val' fields.

  • You can add/remove the columns if required.
  • The Size (y) is updated based on the addition/deletion of the columns.
  • Assign the required value(s) in the '#' and 'Val' fields.
  • Click Update.
